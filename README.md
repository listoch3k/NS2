# NS2
  Скрипт на NS2
Структура проекта:
```
.
├── main.tcl      Основной tcl скрипт с симуляцией на NS2
├── plots     Директория со скриптами gnuplot
│   ├── average_queue.plot      График среднего размера очереди
│   ├── cwnd.plot     График изменения окна TCP
│   └── queue.plot      График моментальной очереди
└── Makefile      Запуск скриптов
```
